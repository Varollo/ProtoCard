# ProtoCard

## About

ProtoCard is a small program to edit and view cards.

The goal is to make it easy for anyone to design a card game, facilitating card creation.

## Getting Started <a name = "getting_started"></a>

You can download the latest build in the **Release** section, or if you want, **clone** the repository and build it yourself using **Unity 6000.2.10f1**.

### Prerequisites

The project is made using Unity version 6000.2.10f1, other than that, no more libraries or programs are required.

If you are just downloading the release version, make sure your operating system can run it.

### Installing

Run the instalation wizard and follow it's instructions.

When it's done, just run the ProtoCard executable.
